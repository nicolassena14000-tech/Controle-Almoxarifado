# Controle-Almoxarifado
Sistema web completo de gestão de almoxarifado com foco em controle de estoque pelo método PEPS (Primeiro a Entrar, Primeiro a Sair). A aplicação oferece um dashboard moderno e intuitivo com métricas em tempo real, controle de entradas e saídas, acompanhamento de lotes, identificação de itens críticos e visualização analítica do estoque. Desenvolvido com HTML, CSS e JavaScript puro, integrado ao Firebase para persistência de dados, o sistema permite registrar movimentações, monitorar histórico, exportar dados em CSV e visualizar gráficos dinâmicos utilizando Chart.js. Também conta com controle de acesso via login e interface responsiva com design premium voltado para usabilidade e clareza operacional.
